## Setup
The first thing to do is to clone the repository:
```sh
$ git clone https://github.com/MahshadAzizi/mahshad_azizi_achare.git
$ cd mahshad_azizi_achare
```
## How to run the app:
Local
```sh
$ python manage.py migrate
$ python manage.py runserver
```
